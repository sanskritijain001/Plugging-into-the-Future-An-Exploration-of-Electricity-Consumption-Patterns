# ⚡ Plugging into the Future: An Exploration of Electricity Consumption Patterns

## 📌 Project Overview
This project analyzes household electricity consumption patterns using data visualization techniques in Tableau and a Flask-based web interface.

---

## 🎯 Objectives
- Understand energy consumption trends
- Analyze peak hours and weekday vs weekend patterns
- Compare sub-metering data (Kitchen, Laundry, Water Heater)
- Visualize monthly and yearly consumption trends

## 🚀 How to Run
```bash
# Step 1: Clone the repo
git clone https://github.com/sanskritijain001/Plugging-into-the-Future-An-Exploration-of-Electricity-Consumption-Patterns

# Step 2: Install dependencies
pip install flask

# Step 3: Run the app
python app.py

# Step 4: Open browser
http://127.0.0.1:5000
```
